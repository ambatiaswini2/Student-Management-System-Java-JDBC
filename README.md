# Student Management System

## Overview

The Student Management System is a console-based Java application developed using JDBC and MySQL. It allows users to perform CRUD (Create, Read, Update, Delete) operations on student records stored in a MySQL database.

## Features

* Add Student
* View All Students
* Search Student by ID
* Update Student Details
* Delete Student Records
* MySQL Database Connectivity using JDBC
* Exception Handling for Reliable Execution

## Technologies Used

* Java
* JDBC
* MySQL
* Eclipse IDE
* Git & GitHub

## Database Schema

Table: `student`

| Column | Type    |
| ------ | ------- |
| s_id   | INT     |
| name   | VARCHAR |
| age    | INT     |
| course | VARCHAR |
| email  | VARCHAR |

## Project Structure

* Database Connection Module
* Add Student Module
* View Student Module
* Search Student Module
* Update Student Module
* Delete Student Module

## How to Run

1. Install MySQL Server.
2. Create a database named `studentdb`.
3. Create the `student` table.
4. Add MySQL Connector/J JAR to the project build path.
5. Update database username and password in the source code.
6. Run `StudentManagementSYstem.java`.
